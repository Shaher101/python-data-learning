# Python Data Learning

This repository contains practice Jupyter notebooks created as part of my learning journey in data analysis and machine learning using Python.

## Contents
- **NumPy.ipynb**  
  Covers fundamental numerical operations, arrays, and basic computations using NumPy.

- **Pandas.ipynb**  
  Focuses on data manipulation, cleaning, and analysis using Pandas DataFrames.

- **MLT.ipynb**  
  Introduces basic machine learning concepts and simple models for practice and understanding.

## Purpose
The goal of this repository is to build a strong foundation in Python for data analysis and machine learning, and to document my progress through hands-on practice.
